- **Titre du Projet :** ERPConnect – Module intelligent d’aide à la décision intégré à un ERP  
- **Description :** Une solution intelligente qui s’intègre à un ERP open source (Odoo, Dolibarr) pour analyser automatiquement les données de gestion (ventes, stocks, achats) et proposer des décisions prédictives.
Le système utilise l’IA pour détecter les anomalies, prédire la demande, recommander des actions et améliorer la performance globale de l’entreprise.  
- **Technologies Suggérées :** Python, FastAPI, Odoo API, Scikit-learn, TensorFlow, React.js, Power BI, Docker, REST API 
- **Thèmes Principaux :** ERP, Intelligence Artificielle, Aide à la décision, Data Analytics, Automatisation

- **Titre du Projet :** Application de Navigation Multimodale Casa–Rabat–Tanger 
- **Description :** Une application web et mobile qui propose des itinéraires intelligents combinant plusieurs moyens de transport (train, bus, voiture, tram, marche) entre Casablanca, Rabat et Tanger.
Le système utilise des données en temps réel (trafic, horaires, météo) pour recommander le trajet le plus rapide, économique ou écologique, selon les préférences de l’utilisateur.  
- **Technologies Suggérées :** React.js, Node.js, Python, Google Maps API, OpenStreetMap, Firebase, TensorFlow (optimisation IA), REST API 
- **Thèmes Principaux :** Mobilité intelligente, Systèmes de transport, Optimisation d’itinéraires, Smart City, IA de recommandation

- **Titre du Projet :** Plateforme de Gestion des Urgences Médicales Géolocalisées  
- **Description :** Application qui connecte patients, ambulances et hôpitaux en temps réel, optimise les trajets d'urgence via algorithmes, prédit les besoins en ressources médicales et gère la disponibilité des lits.  
- **Technologies Suggérées :** React Native, Spring Boot, Google Maps API, WebSocket  
- **Thèmes Principaux :** Gestion des Incidents majeurs, Smart City, Santé embarquée
